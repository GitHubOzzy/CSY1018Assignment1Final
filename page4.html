External Resources that i used!

MDN Web Docs

By far the most useful resource I found when constructing my webpage was MDN Web Docs. This was an extremely helpful website that contains everything about CSS and HTML that is needed to create a website. It includes examples that you can tinker with to alter them to your needs. All the information is also kept as up to date as possible so all the information is valid. Mozilla has provided a valid and helpful resource that is from a trusted source.

Liz Coulter-Smith and Nile

Another very helpful resource theat i used extensivley throughtout my work were the resources on nile given to us by Liz Coulter-Smith. Liz Coulter-Smith is a tutor at the University of Northampton and teaches Website Development. She was available for consultation and discussion to help on this project. All her information was very helpful and also trusted as a source. Nile also lists all the information/sources with interactive examples and is structured corresponding to the weeks we learnt them for easy navigation.

Codepen.io

Another very helpful website was Codepen.io. This is a website that lets you upload small snippets of code to an online compiler that then lets you edit and alter the code to see the effects in realtime. This is helpful because it gives you access to a wide range of solutions to problems you may have encountered. It is also helpful to change elements in realtime to see the desired effects. When doing the initial design of my website in our group, Codepen was also useful to see different design and layouts that gave inspiration.

CSS Tricks

CSS Tricks is a website that has small guides to problems you will encounter when creating a website for example, a guide to flexbox layout and positioning as well as media query guides. This website was very helpful when i first started off with the positioning of images in flexbox but also the media query examples it showcases.

Placeholder

Placeholder is a website that generates placeholder images for your website of any dimesion. I have used one on the Index page of this website. it is very useful for mockups/wireframes of websites to show where images will go, it also dispalys the dimension of the image in the placeholder image itself.
