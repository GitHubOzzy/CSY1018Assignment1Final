FLEXBOX TERMS

1. Display:Flex

this shows the parent element what type of display style it is going to use

2. Flex Direction

This determines what direction the items in the container will face. Values include row, column etc.

3. Justify-Content

this determines the layout/spacing of items in the container. Values include space-between, space-around etc.

4. Order:

this determines which item comes first in the flexbox layout (negative numbers, neutral then positive numbers)

5. Flex-Grow

This determines how much space the item will take up in its container

6. Flex-Shrink

This determines how much space the item will shrink to. Higher numbers mean MORE shrink not vice-versa

7. Flex-Basis

This determines the initial size of the item, can also be declared using box-sizing

8. Flex:

This is a shorthand way of determining 3 values: flow growth, shrink and basis

CSS STYLINGS

9. Color (accepts only American spelling color :/)

Not to be confused with background-color, this version declares the text color

10. Background-Color

This states the background colour of the container unless used in the body tag(in CSS) which then colours the whole webpage

11. Padding

This determines the padding in ALL directions on the content in the container (too specify a perticular direction use "padding-left" etc.

12. Background-Image

This, similar to background-color, lets you chose an image for your container

13. Width:

This lets your set the width of your container using a % or px/em/rem value

14. Height:

Similar to Width, lets you set the height using a % or px/em/rem value

15. Border:

The border property lets you add a fixed border line around your content/Container. You can also add a colour and thickness

16. Margin:

Similar to Border and Padding, Margin alters the space between the container and edge of the viewport.

17. List-Style-Type

This specifies the type of style the list will have, set to none will just show the hyperlinks with no dot next to it

FONT STYLINGS

18. Font-Family

Here you can declare what fonts you want to use, putting a second font after a comma, lets you declare a back-up font incase the first fails (useful for google fonts)

19. Font-Size

This lets you declare how big the font size is, common values include px, em and rem.

20. Text-Transform

This property lets yo transform your text, for example text-transform: uppercase does exactly what is says on the tin

21. Text-Align

This property aligns your text insde its container, text-align:center aligns the text in the very middle of your container

22. Text-Decoration

This property lets you add decor to your text items, like underlines and dotted

23. Text-Decoration-Color

This builds upon the last one by being able to change the color of the decoration

24. Font-Weight

This allows you change the font visual with values such as Bold and Lighter

25. Overflow-wrap

Overflow-wrap determines if the text content of a box will overflow from the container or not, it accepts values such as normal and break-word

SELECTORS

26. Dot (.) Selector

The dot selector is use to select classes in the CSS

27. No selector

Not declaring a selector means that the CSS will select all nodes with that name

28. Hash (#) Selector

The hashkey means that the CSS will select all the ID's with that name

29. the asterix (*) Selector

The asterix is used as an universal selector meaning it will select all elements, can be used with other selectors e.g. *.Classname

30. Browser Selectors

There are different browser selectors that are useful becuase of the way CSS interacts with different browsers. To select them in CSS we type -webkit- for Chrome -moz-for Firfox -o- for Opera -ms- for IE/Edge

HTML Tags

32. Head Tag

The head tag is always at the top of the page, you use this to store all the data your site will need, e.g. Google Fonts, webpage language etc.

33. Body Tag

This tag is used to define the area that the page content will be stored, inside this tag could be many other tags

34. Header Tag

This tag (not to be confused with head) is generally where the webpage title and possibly navigation will go

35. Nav Tag

Althought the navigation could go in the header, it can be seperate using the nav tag, inside it is usually the UL tag and LI tags

36. Undorder List Tag

The UL tag is used to define where the beginning of your hyperlinks will start and end

37. List Tag

The list tag is where you actually store the hyperlink destination and visible text

38. Footer Tag

The footer is the part of the page where you can put things like contact information/forms and in my case a google map

39. Script Tag

The script tag is where you store JavaScript Code, i have used this only for the Google Maps script

40. h Tags

h Tags are used for structuring content, the h tags have pre-defined stylings on them with h1 being the biggest in size

41. p Tags

The p Tags, similar to h Tags, are used for structuring text and also have pre-defined stylings on them

42. Title Tag

The Title Tag goes in the Head tag and is used to declare the Title of the webpage which is then shown in the Tab in the browser

43. Article Tag

The Article Tag is used to structure content containers on the webpage, they can be stored inside other tags aswell

44. Div Tag

The trusty Div Tag can be used when defining content gets vague for other Tags, as long as you give the ID/Class name an appropriate name, it does not matter.

45. Aside Tag

The Aside Tag is another tag that can be used to structure content on your webpage, generally to the side of another content container

Pseudo Classes

46. :hover

Hover is activated when you hover your mouse over an item, usually it changes colour and may have a transisitional effect

47. nth Child

nth-child means a specifc child of the parent element has been selected for styling

48. First-Child

This means only the first child of the parent element is selected

49. Last-Child

Similar to first-child, only the last child is selected

50. :active

When an element has been activated by the user, this psuedo-class defines what will happen to that element
